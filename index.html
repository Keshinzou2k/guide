üß™ technical test examples

1. Swap two pointer values without third variable

Problem: Swap the values of two pointers (addresses) without using a temporary variable.

Solution (XOR swap):

void swap_pointers(int **a, int **b) { *a = (int*)((uintptr_t)*a ^ (uintptr_t)*b); *b = (int*)((uintptr_t)*a ^ (uintptr_t)*b); *a = (int*)((uintptr_t)*a ^ (uintptr_t)*b); }

Arithmetic (careful with overflow): *a = *a + *b; *b = *a - *b; *a = *a - *b; (cast to uintptr_t).

2. Transform 0x58DA27BC37 ‚Üí 0x73CB72AD85

Problem: Given a 5-byte value, apply nibble swap per byte then reverse byte order.

Steps: Original: 0x58 DA 27 BC 37.
- Swap nibbles in each byte: 0x85, 0xAD, 0x72, 0xCB, 0x73.
- Reverse byte order: 0x73 CB 72 AD 85 ‚Üí 0x73CB72AD85.

Code snippet:

uint64_t transform(uint64_t x) { // assuming x holds 5 bytes in lower 40 bits uint64_t result = 0; for (int i = 0; i < 5; i++) { uint8_t byte = (x >> (8*i)) & 0xFF; byte = ((byte >> 4) | ((byte & 0x0F) << 4)); // swap nibbles result |= (uint64_t)byte << (8*(4-i)); // reverse order } return result; }
